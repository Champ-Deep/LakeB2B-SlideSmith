# Lake B2B Brand Guidelines\
\
Brand guidelines for Lake B2B, enabling creation of on-brand content and materials.\
\
## Brand Overview\
\
**Company:** LakeB2B  \
**Tagline:** ENABLING GROWTH  \
**Industry:** B2B Data Services & Marketing Technology\
\
The name "LakeB2B" symbolizes depth (data reservoirs) and collaborative B2B partnerships.\
\

\
## Logo Rules\
\
- Minimum digital: 180px width\
- Minimum print: 30mm width\
- Clear space: Width of letter 'B' on all sides\
- Approved backgrounds: White, light gray, purple (#6D08BE), navy (#011A6B)\
\
**Never:** Alter colors/fonts, skew/rotate, add effects, place on busy backgrounds, merge with other brands.\
\
## Image Guidelines\
\
Use imagery implying: Human-Technology intersection, Innovation, Data visualization, Digital platforms.\
\
Apply brand color tints (purple, pink, orange neons) and curved line graphics.\
