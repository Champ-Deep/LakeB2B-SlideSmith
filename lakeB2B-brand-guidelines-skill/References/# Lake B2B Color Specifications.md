## Color Usage Rules

### Ratio
- **80% Primary Colors** - Used for main brand elements, logos, key graphics
- **20% Secondary Colors** - Used for accents, backgrounds, supporting elements

### Gradients

Create gradients by blending primary colors:
- **Logo Gradient:** Gold (#FFB703) → Red (#E8033A) → Purple (#6D08BE)
- **Gold-Orange Gradient:** #FFB703 → Lighter orange tone
- **Red-Crimson Gradient:** #E8033A → Deeper red
- **Purple-Blue Gradient:** #6D08BE → Lighter purple/blue

### Format Usage
- **Digital (web, social, presentations):** RGB values
- **Print (brochures, business cards, collateral):** CMYK values

### Accessibility
When using colors for text:
- Ensure sufficient contrast ratio (4.5:1 minimum for body text)
- White text works well on Purple (#6D08BE) and Navy (#011A6B) backgrounds
- Dark text needed on Gold (#FFB703) and lighter backgrounds
